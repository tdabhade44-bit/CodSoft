# Simple Calculator -Task 2

## Description
This project is a simple calculator built using Python.
It performs basic arithmetic operations such as:
Addition, Subtraction, Multiplication, and Division.

The program takes two numbers and an operation choice from the user
and displays the calculated result.

## Technologies Used
- Python
- Conditional Statements
- User Input Handling

## Features
- Supports four basic arithmetic operations
- User-friendly menu
- Handles division by zero
- Clean and simple logic
